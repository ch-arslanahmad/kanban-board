# Kanban Board

A progressive web development project built with vanilla HTML, CSS, and JavaScript. Starting from basics and iterating toward a polished, feature-rich task management tool.

## Purpose

This project is for learning core web development fundamentals through building a real, usable application. Each phase builds on the previous, introducing new concepts and improving the overall experience.

## Project Structure

```
kanban-board/
├── index.html
├── main.js
├── style.css
└── README.md
```

## Phases Overview

### Phase 1: MVP Kanban Board

**Goal**: Build a functional, clean kanban board with basic CRUD operations and drag-drop.

**Deliverables**:

- [ ] Set up project structure (HTML, CSS, JS files)
- [ ] Create 3-column board layout (To Do, In Progress, Done)
- [ ] Implement add task functionality
- [ ] Implement delete task functionality
- [ ] Implement drag-and-drop between columns
- [ ] Apply minimal, clean styling
- [ ] Test all interactions

**Tech Stack**:

- Vanilla HTML, CSS, JavaScript

---

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding tasks!

## Future Phases
