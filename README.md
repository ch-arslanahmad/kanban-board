## Purpose

This project is for learning core web development fundamentals through building a real, usable application. Each phase builds on the previous, introducing new concepts and improving the overall experience.

## Project Structure

```
kanban-board/
├── index.html
├── main.js
├── style.css
└── README.md
```

## Phases Overview

### Phase 1: MVP Kanban Board

**Goal**: Build a functional, clean kanban board with basic CRUD operations and drag-drop.

**Deliverables**:

- [x] Set up project structure (HTML, CSS, JS files)
- [x] Create 3-column board layout (To Do, In Progress, Done)
- [x] Implement add task functionality
- [x] Implement delete task functionality
- [x] Implement drag-and-drop between columns
- [x] Apply minimal, clean styling
- [x] Test all interactions

**Tech Stack**:

- Vanilla HTML, CSS, JavaScript

---

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding tasks!
